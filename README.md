# AI Updater Pro

Telegram bot + server: driver messages → OpenAI → replies in Telegram.

## Deploy on Railway

1. Push **all** files in this folder to GitHub (not only this README).
2. Connect Railway to repo `umarmax/Ai-Updater`.
3. Add variables from `.env.example` in Railway → Variables.

**Non-coder guide:** see `docs/NEXT_AFTER_GITHUB.md`.
